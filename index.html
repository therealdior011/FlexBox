FlexBox
CSS Flexbox — это техника для создания сложной модели макета на основе оси с равномерным распределением элементов по странице.
Flexbox, веб-дизайн jarayoni sodda qiluvchi vositalardan biridir. U sahifa tarkibiy qismlarini joylashtirishni yengillashtiradi.
